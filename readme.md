## Пример написание змейки на JS с Canvas и обернутое в Electron

1. npm install
2. git init
   git add .
   git commit -m "Start"
3. пишем все в main.html - он запускается в браузере - получили змейку
4. создаем файлы
    - package.json
    - main.js
    - scripts - не создавал пока
    - renderer.js
5. npm install electron --save-dev
